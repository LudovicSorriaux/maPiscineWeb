# Préférences de l'utilisateur pour l'agent

Date: 2026-02-10

Préférence principale :
- **Privilégier la réflexion à la vitesse.**

Détails et consignes pour l'agent :
- Avant d'appliquer des changements structurels, prendre le temps d'analyser l'impact global (lire les fonctions appelantes, dépendances et usages transverses).
- Préférer proposer un plan conservateur et réversible (petits patches, tests ciblés) plutôt que des modifications rapides et larges.
- Toujours consigner les décisions importantes dans le repo (`.agent_preferences.md`) et mettre à jour la `TODO` list pour visibilité.
- Demander confirmation pour les changements à fort impact (réorganisation de fonctions, changement d'API interne, suppression de code partagé).

Emplacement : fichier à la racine du workspace `maPiscinev4.5Web-d1_mini`.

Usage : l'agent consultera ce fichier au début des tâches critiques pour rappeler la préférence de l'utilisateur.
